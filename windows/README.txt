# sc2sceneswitcher setup guide

1. Run the configuration utility "configure.bat" (follow the steps as prompted)
2. Run the "run.bat" command


## run on startup

1. Right click the "run.bat" file and select "Show more options" then click "Create shortcut"
2. Press "Windows Key + R"
3. Type "shell:startup" and press enter
4. Move the "run.bat" shortcut file to the startup folder that opens


## modify configuration

Either:

Run the "configure.bat" script again

OR

Manually edit the "sc2sceneswitcher.ini" file
